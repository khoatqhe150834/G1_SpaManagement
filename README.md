# Spa Management System

A web-based application for managing spa services, appointments, and customer information.

## Project Structure

```
SpaManagement/
├── src/
│   ├── main/
│   │   ├── java/        # Java source files
│   │   ├── resources/   # Configuration files
│   │   └── webapp/      # Web resources
│   └── test/            # Test files
└── pom.xml              # Maven configuration
```

## Technologies Used

- Java 11
- Jakarta EE
- MySQL
- Maven
- JUnit
- Lombok

## Setup Instructions

1. Clone the repository
2. Configure MySQL database
3. Update database connection settings in `src/main/resources`
4. Build the project using Maven
5. Deploy to your application server

## Development

- JDK 11 or higher required
- Maven 3.6 or higher required
- MySQL 8.0 or higher required
